## 项目介绍

ssm的校园网上订餐系统。

后端使用Java+ssm+MyBatis+MySQL+shiro

前端使用jsp

1. 前台模块
前台主要功能有：用户注册、用户登录、我的购物车、我的订单、商品评论、校园资讯、修改密码
